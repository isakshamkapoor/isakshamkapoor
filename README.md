## Featured Projects

### DocuMind — RAG-based Document Q&A

A Retrieval-Augmented Generation (RAG) service for chatting with your documents. I built the full pipeline end to end — document ingestion, sentence-aware chunking, embeddings, cosine-similarity vector search, and grounded answer generation with source citations. Ships with pluggable AI backends (OpenAI / Gemini / a fully-offline mode), a FastAPI REST API and web UI, 16 automated tests, Docker, and CI.

### ShortLink — Scalable URL Shortener with Analytics

A production-style URL shortener (Bit.ly–like) demonstrating core backend and system-design ideas: Base62 short codes, LRU caching on the redirect path, token-bucket rate limiting, idempotent shortening, click analytics, and a pluggable storage layer (in-memory + SQLite). Built on FastAPI, with 27 tests across both storage backends, Docker, and CI.

### News Headline Classifier

BERT-based news categorization with optimized preprocessing and low-latency inference.

### Speech CAPTCHA Model

ASR-driven audio CAPTCHA system for accessible, voice-based user verification.

### Image Captioning Generator

CNN+LSTM image captioning pipeline with BLEU-based evaluation.

### Emotion-Based Music & Movie Recommender

Component-based frontend for emotion-driven music and movie recommendations with face-detection input.
